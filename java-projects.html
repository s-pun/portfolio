JAVA PROJECTS

ONLINE COMPUTER SHOP

Eclipse | Java

I programmed this simulation of an online computer shop using Eclipse and Java. This is a non GUI application, so all interaction with the programme is through a text based menu which gives users the choice of selecting various commands such as:
-Create a new computer to be placed in stock.
-Create a new customer looking for a computer.
-Search and match the needs of a customer with computers in stock.
-List available computers in descending order of price.
Every customer and computer in the programme is stored in an array list. Each characteristic of a computer and the preference of a computer (e.g.CPU, RAM, Hard drive) is taken as an input from the user.
The given code snippet creates a menu for the user.

DICE GAME

Eclipse | Java

I programmed this game to learn about Java Swing. It is a simple programme which simulates a game of dice. In this game, the player bets a certain amount of money and gets triple the amount if he wins. The rule of the game is that a player wins a game only if he rolls either 7 or 11 on the dice.

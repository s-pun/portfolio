C# PROJECTS

HANGMAN

Visual Studio 2010 | C#

Hangman is a very popular game that we used to play in our childhood. I programmed this version of hangman as a Visual Studio Windows Forms Application using C#.
In this app, the player has to guess the name of an animal. In the example shown, the word to be guessed has nine letters. The player has to guess each letter by clicking on the alphabet buttons and he gets a maximum of seven chances to guess a wrong letter. After each wrong guess, a head, a body, left arm, right arm, left leg and right leg is drawn to complete a hanging man. When the hanging man is complete, a message is displayed to let the player know that he has lost the game.
If the player succeeds to guess the correct word, a message is displayed to let the player know that he has won.
